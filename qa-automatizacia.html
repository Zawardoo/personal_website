About

Testing fintech since 2022.

From manual QA to full automation — I build the frameworks, pipelines, and performance tests that keep banking systems running.

About me

Currently an Automation Engineer at 365.bank in Bratislava. I design and maintain backend test frameworks in Python with Robot Framework, integrated with Oracle DB, MSSQL, and Apache JMeter.

Before that I spent 2.5 years as a QA Tester at the same bank — testing SOAP/REST APIs, writing regression scripts in SoapUI and Postman, and mentoring 3 junior testers through onboarding.

Outside of work I run a Docker-based homelab, daily-drive Linux, and build side projects with FastAPI, PostgreSQL, and Tailwind CSS.

Test Automation

Backend and mobile test frameworks with Python, Robot Framework, Selenium, Playwright, and Appium — with parallel execution on physical devices via Selenium Grid.

API Testing

SOAP and REST API testing with SoapUI and Postman. Automated regression scripts, contract validation, and synthetic test data generation.

CI / CD

Jenkins pipelines for automatic smoke and regression test runs on every deploy. Git, Gitea, and Docker-based workflows.

Performance Testing

Load and stress testing with Apache JMeter. Results analysis from DB logs and Dynatrace. Health checks and monitoring for critical banking services.
